CS 491 Capstone Project


My capstone project for CS491 at NJIT was to build an ecommerce website for my sponsor, Better Body Club. The website is a custom print store for a various range of products from apparel to buisness cards. This project was my first experience working on a real project as a team with 4 other students. The project came with several challenges and was a valuable learning experience for project management.
